Maua
====

Grockit 2.1 - AngularJS-CoffeScript-Jade-Browserify

## Install 

    $ npm install
    $ npm install gulp-header
    $ bower install

## Run (locally)

    $ gulp -w
    $ Go to http://localhost:8080

## Build 

    $ gulp for building to the build directory
    $ gulp --release for building to the release directory
