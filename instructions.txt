Install: npm install -g generator-angulpify
sudo npm install
sudo npm install gulp-header


Run:
gulp for building to the build directory
gulp --release for building to the release directory
Go to http://localhost:8080
Enjoy!



Vagrant

Install virtual box
Install vagrant
vagrant init ubuntu/trusty64
vagrant up
vagrant ssh
Create user “grockit” if not exists: sudo adduser newuser
Add  grockit to sudoers: sudo addgroup grockit sudo
If there are some problems with locale add this to /etc/profile
  export LC_ALL=en_US.UTF-8  
  export LANG=en_US.UTF-8
ln -s /Users/grockit/workspace ~/workspace
